# 静的アセット (Vite publicDir)

このフォルダーのファイルはビルド時にそのままレンダラー出力へコピーされます。

## ログイン画面の背景画像

ログイン画面の背景にしたい画像を、このフォルダーに

```
public/login-background.jpg
```

として配置してください（`.jpg` 推奨。別形式を使う場合は `index.html` の
`#login-screen-bg` の `src` を合わせて変更）。

ファイルが無い場合は自動的に `<img>` が非表示になり、CSS のグラデーション背景に
フォールバックします（アプリは画像なしでも正常に動作します）。
