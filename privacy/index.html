Mason Launcherが取り扱う情報、保存場所、外部サービスとの通信について説明します。

最終更新日: 2026年6月14日
内容 取り扱う情報 利用目的 保存方法 外部サービス 情報の削除 お問い合わせ

Mason Launcher（以下「本アプリ」）はMinecraft: Java Edition向けの 非公式ランチャーです。本アプリはMicrosoft、Mojang Studios、 Modrinthの公式製品ではありません。

01

取り扱う情報

本アプリは、機能提供に必要な範囲で次の情報を取り扱います。

  • Microsoftアカウント認証トークン
  • Minecraftプロフィールのユーザー名、UUID、スキン情報
  • Minecraft: Java Editionの所有権確認結果
  • ランチャー設定、インスタンス設定、導入済みMODの情報
  • Minecraftおよびランチャーの診断ログ
02

利用目的

これらの情報は、Microsoftアカウントでのログイン、Minecraftの所有権確認、 ゲームの起動、ゲームファイルとMODの取得、設定保存、 および不具合診断のために使用します。

03

保存方法

認証キャッシュは、利用可能な環境ではOSが提供する暗号化機能を利用して 端末内に保存します。プロフィール、設定、インスタンス、ログも原則として 利用者の端末内に保存されます。

access token、refresh token、Authorizationヘッダーなどの秘密情報は、 ランチャーの診断ログでマスクするよう設計されています。

独自の収集サーバーについて

現在、本アプリ独自のサーバーへ利用者情報や利用状況を送信する テレメトリ機能は実装していません。

04

外部サービスへの通信

本アプリは機能提供のため、主に次の外部サービスと通信します。

  • Microsoft Entra ID、Xbox Live、XSTS、Minecraft Services
  • Mojang / Minecraftの公式メタデータ・配信サービス
  • Modrinth APIおよびModrinth配信サービス
  • Javaランタイムの公式配信サービス

各サービスでの情報の取り扱いには、それぞれの利用規約と プライバシーポリシーが適用されます。

Microsoft Privacy Minecraft EULA Modrinth Privacy
05

情報の削除

本アプリからログアウトすると、保存済みのMicrosoft認証キャッシュ、 Minecraftプロフィール、および認証済みオフライン起動権限が削除されます。 設定、インスタンス、ゲームデータは、利用者が端末上のファイルを削除することで 消去できます。

06

第三者への提供

開発者が利用者の個人情報を販売することはありません。 認証やダウンロードなどの機能実行時には、 必要な情報が前述の外部サービスへ直接送信されます。

07

声明の変更

本声明は、本アプリの機能変更などに応じて更新される場合があります。 更新時は、このページの最終更新日を変更します。

08

お問い合わせ

本声明または本アプリについての問い合わせは、 Mason LauncherのGitHub Issuesからお願いします。
