プライバシーに関する声明

Mason Launcherが扱う情報、保存場所、外部サービスとの通信について説明します。 本ページは公開用の簡易声明であり、機能追加に合わせて更新されます。

Mason Launcher（以下「本アプリ」）はMinecraft: Java Edition向けの非公式ランチャーです。 本アプリはMicrosoft、Mojang Studios、Modrinthの公式製品ではありません。

取り扱う情報

本アプリは、機能提供に必要な範囲で次の情報を扱います。

Microsoftアカウント認証情報
access token、refresh tokenなど。ログには出さない設計です。
Minecraftプロフィール
ユーザー名、UUID、スキン情報、所有権確認結果。
ランチャー設定とインスタンス情報
プロファイル、Java設定、導入済みMOD、起動ログ。

利用目的

これらの情報は、Microsoftアカウントでのログイン、Minecraftの所有権確認、 ゲーム起動、ゲームファイルとMODの取得、設定保存、不具合診断のために使用します。

保存方法

認証キャッシュは、利用可能な環境ではOSが提供する暗号化機能を利用して端末内に保存します。 プロファイル、設定、インスタンス、ログも原則として利用者の端末内に保存されます。

ログの扱い

access token、refresh token、Authorization、XSTS、sessionなどの秘密情報は、 診断ログに出力しない、またはマスクする設計です。

外部サービスへの通信

本アプリは機能提供のため、主に次の外部サービスと通信します。

Microsoft / Xbox / Minecraft Services
ログイン、XSTS、Minecraftプロフィール、所有権確認。
Mojang / Minecraft公式メタデータ
version JSON、client.jar、libraries、assetsの取得。
Modrinth API
MODとModpackの検索、メタデータ取得、ファイル取得。
Microsoft Privacy Minecraft EULA Modrinth Privacy

情報の削除

本アプリからログアウトすると、保存済みのMicrosoft認証キャッシュ、Minecraftプロフィール、 認証済みオフライン起動権限が削除されます。設定、インスタンス、ゲームデータは、 利用者が端末上のファイルを削除することで消去できます。

第三者への提供

開発者が利用者の個人情報を販売することはありません。 認証やダウンロードなどの機能実行時には、必要な情報が前述の外部サービスへ直接送信されます。

声明の変更

本声明は、本アプリの機能変更などに応じて更新される場合があります。 更新時は、このページの最終更新日を変更します。

お問い合わせ

本声明または本アプリについての問い合わせは、Mason LauncherのGitHub Issuesからお願いします。
