遊ぶ環境を、きれいに分ける。

Mason Launcherは、Minecraft: Java Editionの環境をプロファイルごとに整理する非公式ランチャーです。 日本語UI、公式メタデータ方式、Javaランタイム管理、Modrinth連携、起動ログ診断をひとつの画面にまとめます。

最新版をダウンロード ソースを見る

プロファイルをカードで管理

Vanilla、Forge、Fabric、NeoForgeなどの環境を分けて、起動状態とログを追いやすくします。

01 Official metadata

Minecraft本体やlibrariesは同梱せず、公式URLから検証して取得します。

 02 Instance isolation

mods / config / saves / logsをプロファイル単位で分離します。

 03 Launch diagnostics

Java、classpath、stdout/stderr、latest.logを段階別に記録します。

ランチャーを、環境管理ツールへ。

Mason Launcherは、Minecraftのバージョン、MOD、設定、セーブデータをプロファイル単位で管理します。 共有キャッシュを活用しながら、遊び方ごとの環境を混ぜずに保ちます。

Microsoft認証、Minecraft所有権確認、Java選択、起動引数生成、プロセス起動、ログ保存を段階ごとに扱い、 起動できないときの原因を追いやすくすることを重視しています。

Material 3の面で、機能を読みやすく。

ボタン、カード、リスト、チップ、プログレスはMaterial Webコンポーネントを使い、色はMaterial 3の役割ベースで整理しています。

分離されたインスタンス

MOD、config、saves、logsをプロファイルごとに分け、VanillaとMOD環境の混線を防ぎます。

Javaランタイム管理

Minecraft世代に合うJava majorとarchitectureを確認し、必要なランタイムを選択します。

Modrinth連携

現在のMinecraftバージョンとMODローダーに合うMODやModpackを探し、対象インスタンスへ導入します。

日本語に強いUI

日本語を標準に、英語、繁体字、簡体字、韓国語へ対応します。OS言語の自動検出も扱います。

正規認証と検証

Microsoft認証と所有権確認を分離し、ダウンロードしたファイルはhashとsizeで検証します。

見える起動プロセス

Java、mainClass、classpath、プロセス状態、stdout/stderr、latest.logを追跡します。

起動までの流れを、段階で見せる。

失敗箇所を隠さず、認証・取得・検証・起動をそれぞれ独立した状態として扱います。

認証と所有権確認
Microsoft / Xbox / Minecraft Servicesの段階を分離します。
公式メタデータから取得
version JSON、client.jar、libraries、assetsを検証しながら取得します。
Javaと起動引数を解決
必要Java、natives、classpath、mainClass、JVM/game argsをログ化します。
Minecraftプロセスを起動
stdout/stderr、exit code、latest.log、ウィンドウ確認を追跡します。
