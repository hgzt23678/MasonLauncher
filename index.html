遊ぶ環境を、きれいに分ける。

Mason Launcherは、Minecraft: Java Editionの環境をプロファイルごとに整理する非公式ランチャーです。 日本語UI、公式メタデータ方式、Javaランタイム管理、Modrinth連携、起動ログ診断をひとつのワークスペースにまとめます。

ランチャーを、環境管理ツールへ。

Mason Launcherは、Minecraftのバージョン、MOD、設定、セーブデータをプロファイル単位で管理します。 共有キャッシュを活用しながら、遊び方ごとの環境を混ぜずに保ちます。

Microsoft認証、Minecraft所有権確認、Java選択、起動引数生成、プロセス起動、ログ保存を段階ごとに扱い、 起動できないときの原因を追いやすくすることを重視しています。

必要な機能を、Material 3の面に整理。

カード、リスト、チップ、ボタンはMaterial Webコンポーネントを使い、色はMaterial 3のロールに寄せています。

分離されたインスタンス

MOD、config、saves、logsをプロファイルごとに分離し、VanillaとMOD環境の混線を防ぎます。

Javaランタイム管理

Minecraftの世代に合うJava majorとarchitectureを確認し、必要なランタイムを選択します。

Modrinth連携

現在のMinecraftバージョンとMODローダーに合うMODやModpackを探し、対象インスタンスへ導入します。

見える起動プロセス

Java、mainClass、classpath、プロセス状態、stdout/stderr、latest.logを追跡します。

日本語に強いUI

日本語を標準に、英語、繁体字、簡体字、韓国語へ対応します。OS言語の自動検出も扱います。

正規認証と検証

Microsoft認証と所有権確認を分離し、ダウンロードしたファイルはhashとsizeで検証します。
